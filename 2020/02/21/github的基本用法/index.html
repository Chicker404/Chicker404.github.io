2. 仓库基本管理


code: 仓库中的文件……
issue: 其他人对仓库中项目的建议
pull request: 其他人fork走代码, 对代码进行修改后，向你发起合并到原代码中的请求
setting: 仓库设置信息，删除仓库也在这里
马赛克区域为仓库名，edit修改
commit: 记录对仓库的修改，包括新增、修改、删除文件，可以点进去查看修改或回滚数据
clone or download: 将仓库中的信息下载到本地

本地仓库的创建与初始化

  1. 创建仓库文件夹
  2. 在文件夹打开 git bash，输入git init

Git的基本工作流程

  1. git add: 将文件加载到暂存区
  2. git commit -m “对文件的描述”: 将文件加载到本地仓库
  3. git push: 将文件上传到 github 中的仓库

git status: 查看文件夹中所有文件状态

git config –list查看git匹配的github用户名及邮箱
