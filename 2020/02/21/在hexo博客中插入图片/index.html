在hexo博客中插入图片

Share

step 1

在blog/_config.yml中将post_asset_folder设置为true

step 2

在blog目录下执行：

npm install hexo-asset-image --save

安装了cnpm可以：

cnpm install hexo-asset-image --save

step 3

安装之后在 hexo n 之后会出现一个和md博客同名的文件夹，将需要上传的图片放到里面

step 4

因为图片文件夹名和博客名相同，在想要插入图片的位置使用：

![图片标题](博客名/图片名.后缀)

就可以插入图片了

图片无法显示

上述步骤后无法显示图片，是因为hexo3版本对插件支持有问题，hexo-asset-image插件在处理data.permalink链接时出现路径错误，把年月去掉了，导致最后生成的路径为%d/xxx/xxx。
需要对其做兼容处理，通过判断当前版本是否等于3的版本做不同的路径分割。

打开_hexo-asset-image@1.0.0@hexo-asset-image/index.js
第8行处加入hexo版本：

var version = String(hexo.version).split('.');

修改data.permalink的处理：

var link = data.permalink;  
if(version.length > 0 && Number(version[0]) == 3) 
    var beginPos = getPosition(link, '/', 1) + 1; 
else 
    var beginPos = getPosition(link, '/', 3) + 1;

也可以直接安装修改过的插件：

npm install https://github.com/7ym0n/hexo-asset-image --sa

安装了cnpm：

cnpm install https://github.com/7ym0n/hexo-asset-image --sa

cnpm是淘宝提供的镜像，可以使用cnpm命令行工具代替默认的npm
